LIFO(Last In Fist Out) 스택(배열)

1. 스택 최대사이즈 지정 
#define MAX_STACK_SIZE 100

2. 100의 크기를 가지는 스택 배열 생성
int stack[MAX_STACK_SIZE]

3. 배열 최상단 데이터 위치 지정
int top = -1

4. 스택이 비어있는지 확인하는 함수 생성
	top이 0보다 작으면 1 반환
	그게 아니면 0 반환

5. 스택이 꽉차있는지 확인하는 함수 생성
	top이 최대사이즈-1 보다 크면 1 반횐
	그게 아니면 0 반환

6. 스택에 값 넣는 함수 생성
	4번함수 반환값이 1이면 문구 실행
	그게 아니면 값을 1 증가한 스택 배열에 저장

7. 스택에 값 빼는 함수 설정
	5번 함수 반환값이 1이면 문구 실행
	그게 아니면 해상 스택 배열에서 1 감소
